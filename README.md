# 🛒 Amazon Clone

A responsive Amazon-inspired web application built while following the **SuperSimpleDev JavaScript Course**. After completing the tutorial, I revisited the entire codebase to understand how each module works, how data flows through the application, and how different JavaScript concepts come together in a real project.

__The project serves as a practical learning milestone in my frontend development journey rather than an original product.__

__Although the implementation follows the course, I revisited the code independently to understand the architecture, execution flow, and core JavaScript concepts before deploying the project.__
